# Zecpath AI System

## Objective
This project is designed to build an AI-powered recruitment and candidate screening system with modular architecture.

---

## Project Structure

- data/ → stores datasets and sample files
- parsers/ → resume and document parsing modules
- ats_engine/ → ATS matching and filtering system
- screening_ai/ → candidate screening AI models
- interview_ai/ → AI interview assistance modules
- scoring/ → candidate scoring and ranking logic
- utils/ → utility functions and logging setup
- tests/ → testing scripts and validation
- logs/ → application log files

---
